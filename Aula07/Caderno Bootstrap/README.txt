INSTRUÇÕES (offline):
 - Os arquivos usam CDN para carregar Bootstrap (CSS/JS) por padrão.
 - Para uso totalmente offline, faça o download manual dos arquivos compilados do Bootstrap (CSS e JS)
   a partir de https://getbootstrap.com/docs/5.3/getting-started/download/  (procure pelos arquivos 'bootstrap.min.css' e 'bootstrap.bundle.min.js')
 - Salve esses arquivos na pasta 'assets/' e atualize os links nos HTMLs (substitua as URLs CDN por 'assets/bootstrap.min.css' e 'assets/bootstrap.bundle.min.js').
 - Se quiser, diga que eu mesmo incluo os arquivos do Bootstrap no pacote (posso tentar adicioná-los aqui se você quiser).
