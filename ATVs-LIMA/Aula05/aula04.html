Dicas para usar CSS

  • Use seletores de classe para aplicar estilos a múltiplos elementos.
  • Utilize seletores de ID para estilos únicos.
  • Prefira classes para reutilização de estilos.
  • Organize seu CSS em arquivos externos para melhor manutenção.
  • Teste diferentes propriedades e valores para ver o efeito visual.

O que é id e class

O atributo id é usado para identificar um elemento único na página. Ele deve ser único dentro do documento HTML.

O atributo class é usado para classificar elementos em grupos. Vários elementos podem compartilhar a mesma classe.
