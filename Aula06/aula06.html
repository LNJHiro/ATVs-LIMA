Introdução ao Modelo de Caixas

Em CSS, elementos HTML funcionam como caixas (containers). Entender o modelo de caixas facilita a criação de layouts previsíveis. O box model determina como largura e altura são calculadas e como cada elemento "ocupa" espaço na página.

Observação comum: Muitos navegadores aplicam margin: 8px ao body por padrão. Aqui removemos para demonstrar controles explícitos.

Anatomia de uma Caixa

Uma caixa é composta por (de dentro para fora):

  1. Content — área do conteúdo (texto, imagem) (width/height aplicam-se ao conteúdo quando box-sizing: content-box).
  2. Padding — espaço entre conteúdo e borda (transparente).
  3. Border — borda ao redor do padding (ocupa espaço, pode ter espessura e estilo).
  4. Outline — traço fora da borda, não ocupa espaço no fluxo (útil para acessibilidade/ foco).
  5. Margin — espaço externo que separa a caixa de outras caixas (transparente).

Representação visual (didática)

content (300 × 50)
margin: 50px (área externa visível no demo) outline: 30px (simulado com box-shadow) border: 10px solid red padding: 20px content: width=300px height=50px

Cálculo do tamanho total (largura):
margin-left + outline-left + border-left + padding-left + width + padding-right + border-right + outline-right + margin-right
= 50px + 30px + 10px + 20px + 300px + 20px + 10px + 30px + 50px = 520px

Cálculo do tamanho total (altura):
50px + 30px + 10px + 20px + 50px + 20px + 10px + 30px + 50px = 320px

Tipos de Caixa: Block-level vs Inline-level

Block-level (ex.: div, section) ocupa a largura disponível e quebra linha. Inline-level (ex.: span, a) ocupa apenas o necessário e não força quebra.

inline element outro inline um mais
agora um elemento em display:block ocupa toda a largura disponível

Exemplo: box-sizing

Por padrão o box-sizing é content-box (width/height aplicam-se só ao conteúdo). Usando box-sizing: border-box a largura inclui padding e border (facilita layouts).

content-box: width=200px (+ padding + border)
box-sizing: content-box;

Aqui a largura total será maior que 200px por causa de padding/border.

border-box: width=200px (inclui padding/border)
box-sizing: border-box;

Aqui a largura total será exatamente 200px — padding/border estão dentro.

Resumo / Dicas rápidas

  • width/height referem-se ao content quando box-sizing: content-box.
  • Para layouts previsíveis use box-sizing: border-box na raiz.
  • Outline não ocupa espaço — útil para foco acessível.
  • Use shorthands para código mais limpo (padding, border, margin).
  • Centralize blocos com largura fixa: margin: 0 auto;.
